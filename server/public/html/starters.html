Overview

AI Tourney is designed to help programmers learn about artificial intelligence (AI). There are already many resources available that document the theory of AI. Instead, AI Tourney acts as a fun place to practice programming AI and gain practical experience.


Choose a game

To get started, choose a game to build an AI bot for. If you are new to creating AI, try starting with one of the simpler games (ie. mancala or farkle). Don't worry, just because the game is simple does not mean your AI must be!

Once you have chosen a game, take some time to learn about it. You can try playing the game as a human (/play). You'll want to read up on the game's documentation and rules as well. The AI program will need to be able to understand the messages sent by the game. These messages represent the game's state, which is encoded in JSON.


Download client

Before you can build a bot, make sure to download the AI Tourney client application. This program runs on your computer and connects to the AI Tourney server.

Create a bot

It's time to start programming! You can program in whatever language you want. The only requirements are that you must be able to compile an executable and parse JSON.

Here is some pseudocode of what basic bot code could look like:

var data = JSON.parse( system.args[1] );   // Parse game state (from command line args)
                          // into a variable
var move = getMove( data.state );       // Create the next move
                           // getMove() contains most of the AI code
print( move );                   // Client sends printed move

In short, the bot code needs to parse the game state (JSON arguments), create a move, and print that move. Once your bot can do this, compile an executable.

Note, there is not currently a limit for your bot to take its turn. However, for now, please limit your computation time to ~1.0 seconds. Thanks!


Compete

Now start up the client application. Enter your login credentials, select the game you are playing, and select your bot executable. When you are ready, press start.

AI Tourney will automatically pair you up with a default bot. Once a game is started, you and your opponent will take turns making moves. When it is your turn, the client app will run your executable with the current game state as the command line argument. Once the executable prints your move, the client will then send the move to AI Tourney.

If you wish to watch the game, go to /watch. Enter your gameID and press "watch" to see the game in progress. Your gameID will be listed at the bottom of the client application.
