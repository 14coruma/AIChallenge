§ Warring Kingdoms


Overview

You are a ruler of a kingdom during the heart of the middle ages. To your dismay, a rival kingdom has popped up surprisingly close to yours. You must prove to the world that your kingdom is the greatest!

There are two ways to win Warring Kingdoms. The kingdom with the most collected food at the end of 500 turns wins. However, you can also win by razing the opponent's keep.

How to play (as a human)

Movement and attacking:

• Click on the unit you wish to move. The unit should become highlighted by a red cursor.
• Next click on a target.
• Then click on the move button located on the control panel at the right side of the screen.
• If you wish to attack the target, click on the attack button instead.

Farming:

• Farming is done by your farmer automatically.
• If your farmer dies, hopefully you have enough food to train another!

Training units:

• If you have enough money to train a unit, the unit's icon will appear on the control panel.
• Click this icon to train the unit. It will appear next to your keep.

Building:

• To build a wall, first click on the unit you want to use.
• Then click the target location. This must be directly adjecent to your unit.
• Finally, click on the build icon on the control panel.


The game state

Like all games at AI Tourney, the game state is stored in JavaScript Object Notation (JSON). This is what the farkle JSON looks like:

{
 "id": (number), // The gameID
 "game": "farkle",
 "players": [
  {
   "username": (string)
   "score": (number) // The player's total score
   "fail": (0 or 1) // 1 if a player makes an illegal move
  },
  { ... } // Same object for player two
 ],
 "dice": [ // The numbers of the dice that were rolled
   3, 1, 6, 4 // Example of what dice could look like
 ],
 "bank": [ // The numbers of the dice in the bank
   1, 5 // Example of what a bank could look like
 ],
 "temp": (number), // Temp score of dice (if player has score banked up)
 "currentPlayer": (0 or 1), // Whether it is player0's or player1's turn
 "gameOver": (0 or 1), // Whether the game is over (1) or not (0)
 "winner": (-1 or 0 or 1), // The winner player (-1 for no winner yet)
 "error": (string), // If any error occured it should show up here
}

Your move

The move in farkle should be a JSON (formatted as a string). Your bot executable should print a JSON string to the command line to make a move. Here is what the move JSON should look like:

{
 "bank": [ 1, 5 ], // The dice #s that you want to put in the bank,
 "done": (0 or 1), // If you want to keep rolling (1) or not (0)
}
